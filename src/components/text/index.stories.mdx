# Text

Text components are defined by their `variant` and `color` properties. The default variant is `body1` and the default color is inherited.

<Text>This text itself is using the defaults!</Text>

You can also use an `as` prop to render the component with the _appearance_ of a `variant` but the _identity_ of something else.

You might want to render a `span` with big text, similar to the `h1` variant.

<Text variant="h1" as="span">
  Special span!
</Text>
